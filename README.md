# AI Voice Assistant
Project for 2024-Q1 ConsoleDot Hackathon

## Requirements
- Python 3.8 or later
- PyAudio
- SpeechRecognition
- gTTS
- playsound
- openai

Requires `.env` file with the following keys:
- `OPENAI_API_KEY` - Your OpenAI API key
